# workstation-server

This scripts generates a list of packages you should remove in workstation version so that it can be with the same packages of the server version.
After doing this, the whole GUI (Gnome/ X Server/ applications/ etc) will be removed and this will free some disk space and memory.

For example, see this before and after:

- Original Workstation version 7: 
    - Initial RAM: 363 MB
    - Initial Disk: 7.0 GB
- Modified Workstation version 7 (after cleaning up packages):
    - Initial RAM: 111 MB
    - Initial Disk: 3.9 GB


### Generating Package List

Just run in both fresh versions you just installed (example for version 7).

- On Workstation machine:

    ```
    rpm -qa --qf "%{NAME}\n" > redhat7-workstation-packages.txt
    ```

- On Server machine:

    ```
    rpm -qa --qf "%{NAME}\n" > redhat7-server-packages.txt
    ```

Some examples are provided in this folder.

### Running the Script

Ensure both package list files are in the same location of this script and run it.

```
/generate.py
```
